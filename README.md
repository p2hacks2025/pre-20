# P2HACKS2025 アピールシート

## プロダクト名
金塊発掘テトリス

## コンセプト
テトリスのブロックに金塊やプラチナ塊などの貴金属を埋め込みそれを消すことでテーマ「キラキラ」を表現しようとしたゲームです。

## 対象ユーザ
大学生・社会人
日頃のストレスを解消したり、何も考えずにゲームで爽快感を得たい人向け
年齢制限はなく、大人も子供も楽しめるようなゲーム

## 利用の流れ
プロダクトはエンターキーでゲームスタート。ブロックの中に”金塊”や”プラチナ塊”など特殊なブロックが入っており、一行そろえると、ドリルを使って発掘し、お金をもらってドリルの強化、アイテムの獲得を行っていくゲームです。時間経過に従ってブロックの落ちるスピードが速くなるので詰まないようにどんどんブロックを消していきます。それ以上ブロックを積めなくなったらゲームオーバーです。
細かい仕様としては、金塊やプラチナ塊を掘ると普通のブロックだけが積み上がっていってしまうので、お金を多く使うことで、普通のブロックも消すことができるアイテムを得られます。

## 推しポイント
推しポイントは”爽快感”です。金塊やプラチナ塊を発掘したときに鳴る音が爽快感を生み出します。普通のテトリスのように積み上げて消すことが目的ではなく、金塊をたくさん発掘していくことが目的なので、誰でもうまくプレイできる難易度なのも推しポイントです。積み上げたものを壊すという視覚的な爽快感と、壊したときに鳴る気持ちのいい音という聴覚的な爽快感を足し合わせることで、中毒性を生み出しています。他のゲームと差別化している部分は「テトリス」と「発掘」を掛け合わせているという部分です。単に隙間が空かないようにブロックを配置するのではなく、どのように回転させて、金属ブロックを置くのか？、どのタイミングでドリルを使うのか？など、プレイヤーに多くの選択を持ちかけることで、ゲーム性を高めていると言えます。

### 開発における工夫した点
チーム開発における工夫としては役割分担を明確にし、誰が何を担当しているのかを逐一共有し合いました。毎日のミーティングには全員が必ず参加し、次の作業の流れを共有して進めました。
スケジュール管理の工夫としては、P2HACKS2025が始まる前からメンバーの一週間の予定を共有し、一人一人がこなせる範囲の作業を効率的に分担しました。
技術的な工夫としては、コメントや命名規則を統一し、他のメンバーがコードを読みやすい状態を保ちました。何度もテストプレイをし、操作性や機能面で気になる部分があれば、その都度どのように改善すればよいのかを話し合いながら制作を進めました。
